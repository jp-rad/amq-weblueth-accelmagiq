# @weblueth/accelmagiq

Web Bluetooth react accelmagiq service.

[Live Demo (GitHub Pages)](https://jp-rad.github.io/amq-weblueth-accelmagiq/)

## Installation

```bash
npm install @weblueth/accelmagiq --save-dev
```

or

```bash
yarn add --dev @weblueth/accelmagiq
```
